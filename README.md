# Pathways Dashboard

Alumni tracking dashboard for CA STB pathways program.

## Development

```bash
npm install
npm run dev
```

## Deployment

This project is configured for deployment on Vercel.

## Data Updates

To update the data:
1. Replace `public/data/standardized.csv` with new data
2. Commit and push to trigger automatic deployment

## Features

- Total activity distribution (pie & waffle charts)
- Cohort breakdowns  
- Studying qualifications analysis
- Alumni growth projections
- Responsive design
